# Image Asset Extraction

Use only images from approved user-provided files or explicitly permitted public sources.

For PDF pages, prefer extracting meaningful embedded photos. If cropping is not reliable, render the full page or slide as an image and record that choice in the audit.

Save web assets under `public/assets/pdf-images/` with clear lowercase names. Track each asset with source file, page, and confidence.

Never fill gaps with AI images, stock photos, placeholder image services, or unrelated web images.
