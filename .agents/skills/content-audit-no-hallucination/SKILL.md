# Content Audit No Hallucination

Treat source fidelity as a release requirement.

Before finalizing:
- Search the project for placeholder text, sample data, stock image URLs, AI image references, and unsupported claims.
- Confirm every public data item has a source file, page, and confidence value.
- Move low-confidence or ambiguous facts to `DATA_AUDIT.md`.
- Use neutral fallback labels such as "Dang cap nhat", "Lien he tu van", or "Bao gia theo nhu cau khach hang" when the source is incomplete.

Write a concise audit stating what was used, what was excluded, whether external sources were reachable, and whether any outside imagery or fabricated data was used.
