# Performance Accessibility Check

Run available verification commands such as build and lint. Inspect the rendered page when practical.

Review:
- Semantic landmarks and heading order.
- Image dimensions, alt text, lazy loading where appropriate.
- Color contrast and focus-visible styles.
- Broken links, missing assets, console errors, and empty states.
- Bundle warnings that indicate unnecessary heavy dependencies.

Fix issues that block normal use before final reporting.
