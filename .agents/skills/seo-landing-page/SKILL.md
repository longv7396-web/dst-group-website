# SEO Landing Page

Use one clear H1 per page or app, descriptive section headings, useful metadata, and semantic landmarks.

Keep copy credible:
- Put factual claims only where sources support them.
- Prefer service-category language over unsupported outcome promises.
- Use concise CTA text and contact pathways.
- Add Open Graph and description metadata when the stack supports it.

Do not keyword-stuff or invent proof points.
