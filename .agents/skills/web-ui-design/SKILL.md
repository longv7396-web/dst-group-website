# Web UI Design

Build the real product experience first, not a placeholder landing page.

Use the client's brand assets, source imagery, and verified content. Keep visual decisions grounded in the domain, audience, and source material.

For premium agency sites:
- Lead with a strong hero, clear CTA, and immediate brand signal.
- Use confident typography, restrained color, generous spacing, and crisp section rhythm.
- Use cards for repeated items only; do not nest cards.
- Use real images from approved sources; never use stock, AI, or random placeholder assets when source imagery is required.
- Keep mobile layouts dense enough to scan and never let text overlap controls or media.

Before finishing, inspect desktop and mobile screens and remove placeholder copy, fake names, stock URLs, broken assets, and awkward wraps.
